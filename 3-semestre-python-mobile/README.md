# 3º SEMESTRE - Python e Mobile

**Carga Horária:** 480 horas (4 meses × 96 horas)

## Visão Geral

O terceiro semestre expande os conhecimentos do aluno para backend mais robusto com Python e Django, além de introduzir desenvolvimento mobile com React Native. Os alunos aprenderão a construir aplicações escaláveis em backend e criar experiências mobile cross-platform.

## Objetivos do Semestre

- Dominar Python e paradigmas de programação
- Desenvolver aplicações backend robustas com Django
- Entender padrões avançados de desenvolvimento
- Criar aplicações mobile cross-platform
- Integrar frontend, backend e mobile
- Implementar arquitetura escalável
- Trabalhar com APIs e integração de sistemas
- Preparar-se para desenvolvimento profissional

## Estrutura de Módulos

### [Mês 13: Python Fundamentos (96h)](./mes-13-python-fundamentos/README.md)
Introdução a Python, paradigmas de programação, estruturas de dados e conceitos fundamentais.

### [Mês 14: Django Framework (96h)](./mes-14-django/README.md)
Django para desenvolvimento web backend, ORM, admin, forms e arquitetura MVC.

### [Mês 15: Integração de Sistemas (96h)](./mes-15-integracao/README.md)
APIs REST, webhooks, mensageria, background jobs e integração com terceiros.

### [Mês 16-18: Super Projeto Mobile (288h)](./mes-16-18-superprojeto-mobile/README.md)
Desenvolvimento mobile completo com React Native (3 meses intensivos).

## Pré-requisitos

- Conclusão completa do 2º semestre (React + Backend)
- Domínio de programação em JavaScript
- Conceitos de OOP
- Familiaridade com APIs REST
- Git e versionamento

## Ferramentas e Tecnologias

### Backend Python
- **Python 3.10+** - Linguagem
- **Django 4.x** - Framework web
- **Django REST Framework** - APIs REST
- **Celery** - Background jobs
- **Redis** - Cache e message queue
- **PostgreSQL** - Banco de dados

### Mobile
- **React Native 0.71+** - Framework mobile
- **Expo** - Tooling para React Native
- **TypeScript** - Tipagem
- **React Navigation** - Roteamento
- **Redux Toolkit** - State management
- **TailwindCSS Native** - Styling

### Ferramentas DevOps
- **Docker** - Containerização
- **Docker Compose** - Orquestração local
- **GitHub Actions** - CI/CD
- **Heroku/Railway** - Hosting backend
- **EAS Build** - Build mobile
