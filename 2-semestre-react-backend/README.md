# 2º SEMESTRE - React e Backend Full-stack

**Carga Horária:** 576 horas (6 meses × 96 horas)

## Visão Geral

O segundo semestre concentra-se na construção de aplicações web completas, combinando React no frontend com tecnologias backend robustas como Node.js, Express e PostgreSQL. Os alunos aprenderão desde conceitos fundamentais de React até a criação de aplicações full-stack production-ready.

## Objetivos do Semestre

- Dominar React e suas principais bibliotecas do ecossistema
- Desenvolver APIs REST com Node.js e Express
- Trabalhar com bancos de dados relacionais (PostgreSQL)
- Implementar autenticação, validação e segurança
- Fazer deploy de aplicações em produção
- Criar projetos full-stack complexos e escaláveis

## Estrutura de Módulos

### [Mês 7-8: React Fundamentos (192h)](./mes-07-react-fundamentos/README.md)
Introdução aos conceitos fundamentais do React, JSX, componentes, estado e props.

### [Mês 9: React Intermediário (96h)](./mes-08-react-intermediario/README.md)
Hooks avançados, Context API, gerenciamento de estado complexo e performance.

### [Mês 10: React Avançado (96h)](./mes-09-react-avancado/README.md)
Padrões avançados, TypeScript com React, testes automatizados e otimização.

### [Mês 11: Node.js e Express (96h)](./mes-10-nodejs-express/README.md)
Criação de APIs REST, middleware, validação, autenticação e boas práticas.

### [Mês 12: PostgreSQL e Deploy (96h)](./mes-11-postgresql-deploy/README.md)
SQL avançado, design de banco de dados, deploy em cloud e CI/CD.

### [Mês 13: Projeto Full-stack (96h)](./mes-12-projeto-fullstack/README.md)
Aplicação completa integrando frontend React com backend Node.js/Express/PostgreSQL.

## Pré-requisitos

- Conclusão completa do 1º semestre (JavaScript Fundamentos e HTML/CSS)
- Conhecimento de ES6+ (variáveis, funções arrow, destructuring, spread operator)
- Familiaridade com Git e terminal
- Entendimento de conceitos básicos de programação

## Ferramentas e Tecnologias

### Frontend
- **React 18+** - Biblioteca UI
- **TypeScript** - Tipagem estática
- **React Router v6** - Roteamento
- **Axios/Fetch** - Requisições HTTP
- **Tailwind CSS** - Estilização
- **Vite** - Build tool

### Backend
- **Node.js 18+** - Runtime JavaScript
- **Express** - Framework web
- **Sequelize/TypeORM** - ORM
- **JWT** - Autenticação
- **Joi/Zod** - Validação

### DevOps
- **PostgreSQL 14+** - Banco de dados
- **Docker** - Containerização
- **GitHub Actions** - CI/CD
- **Vercel/Railway/Heroku** - Hosting

## Metodologia

### Ciclo de Aprendizado
1. **Conceitos Teóricos** (30% do tempo)
   - Videoaulas explicativas
   - Documentação oficial
   - Exemplos práticos

2. **Prática Guiada** (40% do tempo)
   - Exercícios step-by-step
   - Projetos pequenos e focados
   - Code reviews com mentoria

3. **Desafios Independentes** (30% do tempo)
   - Projetos maiores
   - Problemas reais
   - Integração de conceitos

### Ritmo de Estudo
- **4 horas de aula** (ao vivo ou gravada com interação)
- **4 horas de prática** (exercícios e projetos)
- **2 horas de revisão** (documentação, debugging, aprofundamento)
- **1 hora de mentoria/suporte** (dúvidas, code review)

## Avaliação

### Critérios de Aprovação

### Tipos de Avaliação

## Projetos do Semestre

### Projeto 1: Blog Interativo (Mês 7-8)
**Stack:** React + Tailwind CSS
- Listagem de artigos
- Página de detalhe
- Busca e filtros
- Design responsivo

### Projeto 2: API RESTful de Tarefas (Mês 10)
**Stack:** Node.js + Express + PostgreSQL
- CRUD completo
- Validação de dados
- Tratamento de erros
- Testes unitários

### Projeto 3: Dashboard Analytics (Mês 9)
**Stack:** React + Context API
- Gráficos e estatísticas
- Gerenciamento de estado
- Otimização de performance
- Testes de componentes

### Projeto 4: E-commerce Completo (Mês 12-13)
**Stack:** React + Node.js + Express + PostgreSQL
- Autenticação e autorização
- Carrinho de compras
- Processamento de pagamento (simulado)
- Dashboard administrativo
- Deploy em produção
